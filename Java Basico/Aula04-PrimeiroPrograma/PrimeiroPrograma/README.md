# ☕ Java Básico — Estudos e Exercícios

Repositório com exercícios e práticas desenvolvidos durante o curso
de Java básico do professor Gustavo Guanabara — CursoemVídeo.

## 🛠️ Ferramentas
- Java JDK 25
- Apache NetBeans IDE 29

## 📚 Conteúdo
- Aula 01 — Primeiro programa, estrutura básica e System.out.println

## 📁 Estrutura do Repositório

Cada exercício está organizado em sua própria pasta de projeto NetBeans.
Para abrir: File → Open Project → selecione a pasta desejada.

O código fonte de cada exercício fica em:
`NomeDoProjeto → src → nomedoprojeto → NomeDoProjeto.java`
